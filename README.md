## 仓库介绍
本仓库是针对**王道考研数据结构讲义的课后习题**编写的编程实现代码，后期加入历年真题习题实现，核心目标是：
1. 夯实408数据结构核心知识点，通过编程落地理论知识；
2. 提供可运行、易理解的代码示例，帮助同备考408的同学快速上手；
3. 规范代码风格，贴合考研/面试场景的编程习惯。

### 适用人群
- 备战计算机408统考的考研党
- 初学数据结构、需要通过习题巩固的同学
- 希望复盘王道习题编程实现的开发者

### 参考资料
- 王道论坛《2027数据结构考研复习指导》
