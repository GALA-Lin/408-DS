## 常见问题

1. 代码运行报错？
   - 检查编译器版本（推荐 G++），避免语法兼容问题；
   - 确认代码中的数组大小、指针操作是否越界（已做基础防护，但可自行扩展测试）。
2. 是否覆盖所有课后习题？
   - 优先覆盖核心考点习题（如算法设计题），基础选择题的编程验证会选择性实现，持续更新中。
3. 代码可以直接用于考研答题吗？
   - 代码风格贴合考研答题规范，可在此基础上简化（如去掉调试打印、合并冗余注释）后直接使用核心代码。
